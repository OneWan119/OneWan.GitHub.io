    牛顿迭代法

    在 \([a,b]\) 上连续且单调的函数 \(f(x)\)，求方程 \(f(x)=0\) 的近似解。

    \[ \nonumber x_{i+1}=x_i-\dfrac{f(x_i)}{f^\prime(x_i)} \]

    const double eps = 1e-15;
double Sqrt(double n) {
    double x = 1;
    while (true) {
        double nx = (x + n / x) / 2;
        if (fabs(x - nx) < eps) {
            break;
        }
        x = nx;
    }
    return x;
}
